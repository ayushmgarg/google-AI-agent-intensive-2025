\## üìÖ Day 1: Introduction to Agents \& Agentic Architectures



\### Key Concepts



\- \*\*What are AI Agents?\*\* Systems that go beyond simple prompting by incorporating planning, action, memory, and evaluation capabilities

\- \*\*Agent Loop Workflow\*\* The core cycle of perceiving, reasoning, acting, and learning that differentiates agents from static model calls

\- \*\*Single-Agent Architecture\*\* Building autonomous systems that can execute multi-step tasks independently

\- \*\*Refinement Cycles\*\* How agents iterate and improve their outputs through feedback loops

\- \*\*From Prompt to Action\*\* Understanding the progression from basic LLM calls to sophisticated agentic behavior
